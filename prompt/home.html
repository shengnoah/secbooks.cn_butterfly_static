AI 提示词完全教程（文字 / 图片 / 视频）

这是一套面向「从零到可复用」的提示词（Prompt）教程：从基本概念到可直接套用的模板，覆盖 文字写作（含面向小学生的表达）、图片生成、视频故事与分镜。

你将学到什么

  • 把“想法”翻译成模型能稳定执行的 指令 + 约束 + 输出格式。

  • 文字类提示词：写作、改写、总结、教学（面向小学生）等模板与实例。

  • 小说类提示词：不同风格（悬疑 / 科幻 / 奇幻 / 言情 / 武侠 / 历史等）的写法。

  • 图片类提示词：通用结构、负面提示词、不同风格案例。

  • 视频类提示词：故事梗概 → 分镜脚本 → 镜头语言与约束 → 可执行模板。

适合人群

  • 内容创作者：写文章、写小说、写脚本、做分镜。

  • 设计/视觉方向：用 SD/FLUX/MJ 等工具出图，需要稳定复现风格。

  • 产品/运营：写广告文案、短视频脚本、口播稿。

  • 学生/老师：需要把复杂概念讲给小学生听。

使用方式（建议）

  1. 先读「提示词基本概念」建立共同语言。

  2. 根据你的目标选择章节：文字 / 小说 / 图片 / 视频。

  3. 每次只改一个变量：主题、受众、结构、风格、约束、长度……逐步收敛。

  • 06. 实战工作流：迭代、评审、复用与提示词库

约定：本教程把“提示词”视为一个可版本化的需求规格。

  • 文字模型：输出更像“文档/代码/对话”。
  • 图片模型：输出更像“画面”，需要把视觉约束说清楚。
  • 视频模型：输出更像“连续镜头”，需要把时间、运动与连续性说清楚。
