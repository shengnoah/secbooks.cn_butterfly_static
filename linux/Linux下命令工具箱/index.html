byobu高级化的tmux

这两个工具都属于生产力工具，平时使用的概率频繁，用好了日积月累可以节省很多时间，花点时间学习，可以节省很多不必要的时间花费。

摸索对比了byobu和 tmux ，确认是byobu生产性更高，并且支持 tab页，这种生产方式，本身就比tmux高校，而且快捷按键， tmux.是 control + b 命令， byobu是control + a 命令，并且还支持F区的功能快捷按键，比tmux的快捷按键更方便，一个功能有都快捷按键定义。

平时用的比较的终端是Warp和Tabby，因为每个工具对键盘的F功能区上的按键定义不一样，有的时候会出现冲突的，用control + a 命令的形工，与终端软件本身的冲突是最小的。

Tabby操作

shift + F2横切
ctrl + a % 竖切

shift + 上下左右， 移动屏幕。

ctrl + a 上下左右， 移动屏幕。

[F3]ctrl + a n 向前移动tab
[F4]ctrl + a p 向后移动tab

[F6]ctrl + a d 退出
tmux的退出是ctrl + b d
