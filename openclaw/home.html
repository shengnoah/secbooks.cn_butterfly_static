OpenClaw 完全教程

OpenClaw 完全教程

本教程以 Book Layout 的方式组织，适合把 OpenClaw 当作「个人 Agent 平台」持续落地的人：从跨平台部署开始，到 Skills/命令、接入飞书机器人，再到角色/记忆/Agent 的可复用设定，以及常见自动化使用场景。

你将获得什么

  • macOS / Windows（WSL2）两套可复用的部署路径

  • 常用 Skills 的挑选与落地打法（从“能跑”到“稳定”）

  • 最常用命令速查表（onboard / agent / models / skills / plugins / browser…）

  • 飞书机器人接入思路（事件 → OpenClaw → 回消息）

  • 三份可直接复制的设定文档：角色设定、记忆设定、Agent 设定

  • 典型自动化场景拆解：

    • Polymarket 市场赚 USD（以合规与风控为前提）
    • A 股中低频赚取人民币（以数据/策略/执行闭环为主）
    • 操作 GitHub 自动生成网站内容
    • 发布到小红书
    • 自动收集新闻并产出文章（安全漏洞 / 股票财经 / 产品跟踪）

阅读方式

  • 左侧是“本教程的完整目录”，按顺序阅读即可。

  • 每一章都包含：目标、步骤、坑位、可复制的配置/命令片段。

注：教程以“工程可落地”为导向。涉及交易/金融内容仅做自动化与数据流程示例，不构成投资建议。
