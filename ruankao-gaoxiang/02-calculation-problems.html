计算题高频模型：公式、步骤、典型题完整解答

02. 计算题高频模型：公式、步骤、典型题完整解答

目标：把高频计算模型做成“见题就套”的固定流程。

0. 计算题通用写法（强烈建议每题按此落笔）

  • 已知：摘录数据（带单位）

  • 求：题目问的量

  • 公式/方法：写核心公式

  • 计算过程：代入 → 中间量 → 结论

注意：写“过程”能拿步骤分；只写结果风险很大。

1. 挣值管理（EVM）——高频中的高频

1.1 必背量与公式

  • PV（计划价值）：计划到某时点应该完成的工作量对应预算

  • EV（挣值）：实际完成的工作量对应预算

  • AC（实际成本）：实际花了多少钱

派生量：

  • SV = EV - PV（进度偏差）

  • CV = EV - AC（成本偏差）

  • SPI = EV / PV（进度绩效指数）

  • CPI = EV / AC（成本绩效指数）

完工估算（常考两类）：

  • EAC = BAC / CPI（假设未来绩效与当前一致）

  • ETC = EAC - AC（完工尚需估算）

经验：题目没特别说明时，优先用 EAC = BAC / CPI。

1.2 典型题（完整解答）

题目：某项目总预算 BAC=100 万元。到第 6 个月时：

  • 计划完成 60% 工作量（即 PV 对应 60%）

  • 实际完成 50% 工作量（即 EV 对应 50%）

  • 实际已花费 AC=55 万元

求：SV、CV、SPI、CPI，并判断进度/成本状况；按当前绩效估算 EAC。

解答：

  1. 已知：BAC=100 万；PV=60%×100=60 万；EV=50%×100=50 万；AC=55 万

  2. 求：SV、CV、SPI、CPI、EAC

  3. 公式：

  • SV=EV-PV

  • CV=EV-AC

  • SPI=EV/PV

  • CPI=EV/AC

  • EAC=BAC/CPI

  1. 计算：

  • SV=50-60=-10（万元）→ 进度落后

  • CV=50-55=-5（万元）→ 成本超支

  • SPI=50/60=0.8333→ 小于 1，进度绩效差

  • CPI=50/55=0.9091→ 小于 1，成本绩效差

  • EAC=100/0.9091≈110（万元）→ 预计超支约 10 万

结论：进度落后、成本超支；若不纠偏，完工估算约 110 万。

1.3 易错点

  • “完成比例”用于 EV，“计划比例”用于 PV。

  • SPI/CPI 小于 1 表示绩效差（落后/超支）。

  • 单位保持一致（万元/元）并写清楚。

2. 网络计划（CPM / 关键路径）——工期与时差

2.1 核心概念

  • ES/EF：最早开始/最早完成（正推）

  • LS/LF：最迟开始/最迟完成（逆推）

  • 总时差 TF = LS - ES = LF - EF

  • 关键路径：TF=0 的活动链；其长度=项目工期

2.2 典型题（AON，活动在节点，给持续时间与前置关系）

题目：活动如下（单位：天）：

  • A：3（无前置）

  • B：4（前置 A）

  • C：2（前置 A）

  • D：5（前置 B、C）

  • E：2（前置 D）

求：项目工期、关键路径；并求活动 C 的总时差 TF。

解答：

  1. 正推（ES/EF）：

  • A：ES=0，EF=3

  • B：ES=3，EF=7

  • C：ES=3，EF=5

  • D：ES=max(EF_B,EF_C)=max(7,5)=7，EF=12

  • E：ES=12，EF=14

→ 项目工期=14 天

  1. 逆推（LS/LF）（从终点 E 开始）：

  • E：LF=14，LS=12

  • D：LF=LS_E=12，LS=7

  • B：LF=LS_D=7，LS=3

  • C：LF=LS_D=7，LS=5

  • A：LF=min(LS_B,LS_C)=min(3,5)=3，LS=0

  1. 关键路径：TF=0 的活动

  • A：TF=LS-ES=0-0=0

  • B：TF=3-3=0

  • D：TF=7-7=0

  • E：TF=12-12=0

→ 关键路径：A-B-D-E

  1. 活动 C 的总时差：TF_C=LS-ES=5-3=2（天）

2.3 易错点

  • 合并点（多前置）取 max(EF)；分叉点（多后继）逆推取 min(LS)。

  • 题目若从 1 天开始计，也可用 0 开始计；关键是保持一致。

3. 三点估算（PERT）——期望工期与方差

3.1 公式

  • 期望：TE = (O + 4M + P) / 6

  • 标准差：σ = (P - O) / 6

  • 方差：σ²

3.2 典型题（完整解答）

题目：某任务乐观 O=6 天，最可能 M=9 天，悲观 P=18 天。求期望工期 TE 与标准差 σ。

解答：

  • TE=(6+4×9+18)/6=(6+36+18)/6=60/6=10（天）

  • σ=(18-6)/6=12/6=2（天）

4. 风险概率与期望货币值（EMV）——会算也要会选

4.1 公式

  • EMV = Σ(概率 × 影响)

4.2 典型题（决策树口径）

题目：方案 X 成本 20 万，成功概率 0.7，成功收益 50 万；失败概率 0.3，失败损失 10 万（损失视为负收益）。求 EMV。

解答：

  • 成功净收益=50-20=30

  • 失败净收益= -10-20 = -30

  • EMV=0.7×30 + 0.3×(-30)=21-9=12（万元）

注意：题目如果把“成本”已经包含在收益里，就不要重复减。

5. 质量管理常考：缺陷密度 / 抽样 / 控制界限（概念 + 简算）

这类题通常计算不难，但容易在“定义”上扣分。

  • 缺陷密度：缺陷数 / 规模（如 KLOC、功能点）

  • 控制界限（概念题多）：通常为均值 ±3σ（题目给均值与 σ 时可代入）

6. 成本-收益类：投资回收期 / ROI / NPV / IRR（高频）

6.1 ROI

  • ROI = (收益 - 成本) / 成本

6.2 NPV（净现值）

  • NPV = Σ(现金流_t / (1+i)^t) - 初始投资

考场技巧：把折现系数写出来，分步相加，最后再减初始投资。

6.3 典型题（NPV 完整解答）

题目：某项目初始投资 100 万，未来 3 年每年年末净现金流分别为 40 万、50 万、60 万。折现率 i=10%。求 NPV（保留 2 位小数）。

解答：

  • 年 1 折现：40/(1.1)^1=36.36

  • 年 2 折现：50/(1.1)^2=41.32

  • 年 3 折现：60/(1.1)^3=45.08

  • 现值合计=36.36+41.32+45.08=122.76

  • NPV=122.76-100=22.76（万元）

结论：NPV>0，财务上可行。

7. 计算题“高频清单”速记

  • 挣值：PV/EV/AC → SV/CV/SPI/CPI → EAC/ETC

  • 网络计划：正推/逆推 → TF → 关键路径

  • PERT：TE、σ

  • 风险：EMV、决策树

  • 财务：ROI、回收期、NPV/IRR
