CentOS8下安装Conda的步骤

发表于2024-04-03|更新于2024-04-03|文章
|阅读量:

您可以按照以下步骤在 CentOS 8 上安装 Conda：

  1. 下载 Miniconda 安装包：访问 https://docs.conda.io/en/latest/miniconda.html，下载适用于 Linux 的 Miniconda 安装包。如果您使用的是 64 位系统，则下载 Linux 64-bit 版本。

  2. 安装 Miniconda：在终端中执行以下命令安装 Miniconda（假设下载的文件名为 Miniconda3-latest-Linux-x86_64.sh）：

    bash Miniconda3-latest-Linux-x86_64.sh

    然后根据提示进行安装，一般直接按回车键就行了。安装完成后，要记得打开一个新的终端窗口或者输入 source ~/.bashrc 命令来激活 conda 命令。

  3. 创建虚拟环境：为了隔离不同项目的依赖关系，建议使用虚拟环境进行开发。可以使用以下命令创建一个新的 Python 3.7 环境：

    conda create --name env_name python=3.7

    其中 env_name 是您想要创建的环境名称。

  4. 激活虚拟环境：执行以下命令激活新创建的虚拟环境：

    conda activate env_name

  5. 在虚拟环境中安装软件包：在虚拟环境下使用 conda install 命令安装需要的软件包。例如，要安装 NumPy 和 Pandas，请执行以下命令：

    conda install numpy pandas

    这将在当前虚拟环境中安装 NumPy 和 Pandas 软件包。

注意：以上示例仅供参考，实际应用中可能需要根据具体需求进行修改。
