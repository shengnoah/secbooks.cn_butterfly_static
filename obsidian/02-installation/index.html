1.2 安装与配置

系统要求

Obsidian支持多个平台，系统要求如下：

Windows

  • Windows 10或更高版本

  • 64位系统

  • 至少4GB内存（推荐8GB）

macOS

  • macOS 10.13或更高版本

  • Intel或Apple Silicon处理器

  • 至少4GB内存（推荐8GB）

Linux

  • 64位系统

  • 支持AppImage、Snap或Flatpak

  • 至少4GB内存（推荐8GB）

移动端

  • iOS: iOS 14或更高版本

  • Android: Android 5.0或更高版本

下载Obsidian

官方网站下载

访问Obsidian官方网站下载：

🌐 官方网站: https://obsidian.md

  1. 点击"Download"按钮

  2. 选择你的操作系统

  3. 下载安装包

各平台下载链接

  • Windows: 下载.exe安装程序

  • macOS: 下载.dmg安装包

  • Linux: 下载.AppImage或使用包管理器

  • iOS: 在App Store搜索"Obsidian"

  • Android: 在Google Play搜索"Obsidian"

安装步骤

Windows安装

  1. 运行安装程序

    • 双击下载的.exe文件
    • 如果出现安全提示，点击"运行"

  2. 选择安装位置

    • 默认安装到 C:\Users\用户名\AppData\Local\Obsidian
    • 可以自定义安装路径

  3. 完成安装

    • 等待安装完成
    • 选择是否创建桌面快捷方式

macOS安装

  1. 打开DMG文件

    • 双击下载的.dmg文件
    • 等待挂载完成

  2. 拖动到Applications

    • 将Obsidian图标拖动到Applications文件夹
    • 等待复制完成

  3. 首次运行

    • 从Applications文件夹打开Obsidian
    • 如果出现安全提示，在系统偏好设置中允许运行

Linux安装

使用AppImage

# 下载AppImage文件
wget https://github.com/obsidianmd/obsidian-releases/releases/download/v1.5.3/Obsidian-1.5.3.AppImage

# 添加执行权限
chmod +x Obsidian-1.5.3.AppImage

# 运行Obsidian
./Obsidian-1.5.3.AppImage

使用Snap

sudo snap install obsidian --classic

使用Flatpak

flatpak install flathub md.obsidian.Obsidian

移动端安装

iOS安装

  1. 打开App Store

  2. 搜索"Obsidian"

  3. 点击"获取"下载安装

  4. 等待安装完成

Android安装

  1. 打开Google Play商店

  2. 搜索"Obsidian"

  3. 点击"安装"

  4. 等待安装完成

首次启动

创建或打开仓库

首次启动Obsidian时，你需要选择：

选项1：创建新仓库

  1. 点击"Create new vault"

  2. 设置仓库名称

    • 输入一个有意义的名称，如"我的知识库"

  3. 选择存储位置

    • 选择一个容易找到的位置
    • 推荐：Documents/Obsidian

  4. 点击"Create"

选项2：打开现有文件夹

  1. 点击"Open folder as vault"

  2. 选择包含Markdown文件的文件夹

  3. 点击"Open"

选项3：打开帮助文档

  1. 点击"Open help vault"

  2. 浏览Obsidian官方帮助文档

  3. 学习基础功能

仓库（Vault）的概念

**仓库（Vault）**是Obsidian中的核心概念：

  • 📁 一个仓库就是一个文件夹

  • 📝 包含你的所有笔记和配置

  • 🔧 每个仓库有独立的设置和插件

  • 💡 可以创建多个仓库用于不同目的

基础配置

语言设置

  1. 打开设置

    • 点击左下角的齿轮图标⚙️
    • 或使用快捷键Ctrl/Cmd + ,

  2. 选择语言

    • 在左侧菜单选择"About"
    • 找到"Language"选项
    • 选择"简体中文"
    • 重启Obsidian生效

外观设置

选择主题

  1. 打开设置 → 外观

  2. 选择基础主题

    • Light（浅色）
    • Dark（深色）

  3. 浏览社区主题

    • 点击"管理"按钮
    • 浏览和安装社区主题

调整字体

设置 → 外观 → 字体

  • 文本字体: 正文使用的字体

  • 等宽字体: 代码块使用的字体

  • 字体大小: 调整文本大小

编辑器设置

基础编辑器设置

设置 → 编辑器

推荐配置：

  • ✅ 显示行号: 方便定位

  • ✅ 自动配对括号: 提高效率

  • ✅ 智能缩进: 自动缩进

  • ✅ Vim模式: 如果你熟悉Vim（可选）

文件与链接

设置 → 文件与链接

推荐配置：

  • 新建笔记的存放位置: 选择默认文件夹

  • 新链接格式: 推荐"相对路径"

  • 使用Wiki链接: 开启（Obsidian特色）

  • 自动更新内部链接: 开启

核心插件配置

Obsidian内置了多个核心插件：

启用推荐的核心插件

设置 → 核心插件

推荐启用：

  • ✅ 文件恢复: 自动备份，防止数据丢失

  • ✅ 快速切换: 快速打开文件

  • ✅ 命令面板: 快速执行命令

  • ✅ 模板: 使用模板创建笔记

  • ✅ 日记: 每日笔记功能

  • ✅ 标签面板: 管理标签

  • ✅ 大纲: 显示文档大纲

  • ✅ 反向链接: 显示反向链接

文件夹结构建议

推荐的文件夹结构

我的知识库/
├── 📁 00-收件箱/          # 临时笔记和想法
├── 📁 01-项目/            # 项目相关笔记
├── 📁 02-领域/            # 按领域分类的笔记
│   ├── 编程/
│   ├── 设计/
│   └── 写作/
├── 📁 03-资源/            # 参考资料
│   ├── 书籍/
│   ├── 文章/
│   └── 课程/
├── 📁 04-归档/            # 已完成的内容
├── 📁 附件/               # 图片、PDF等附件
└── 📁 模板/               # 笔记模板

创建文件夹

  1. 在文件浏览器中右键

  2. 选择"新建文件夹"

  3. 输入文件夹名称

  4. 按Enter确认

同步配置

官方同步服务

Obsidian Sync ($10/月)

优点：

  • ✅ 端到端加密

  • ✅ 版本历史

  • ✅ 自动同步

  • ✅ 支持所有平台

设置方法：

  1. 订阅Obsidian Sync

  2. 设置 → Sync

  3. 登录账号

  4. 选择要同步的仓库

第三方同步方案

iCloud（macOS/iOS）

  1. 将仓库放在iCloud Drive

    ~/Library/Mobile Documents/iCloud~md~obsidian/Documents/

  2. 在iOS设备上打开同一仓库

Dropbox

  1. 将仓库放在Dropbox文件夹

  2. 在其他设备上打开同一仓库

  3. 注意: 避免同时编辑同一文件

Git（高级用户）

使用Git进行版本控制和同步：

# 初始化Git仓库
cd /path/to/vault
git init

# 添加.gitignore
echo ".obsidian/workspace*" >> .gitignore
echo ".obsidian/cache" >> .gitignore

# 提交更改
git add .
git commit -m "Initial commit"

# 推送到远程仓库
git remote add origin <your-repo-url>
git push -u origin main

推荐插件：Obsidian Git

Syncthing（开源方案）

  1. 安装Syncthing

  2. 配置同步文件夹

  3. 在多个设备间同步

优点：

  • ✅ 完全免费

  • ✅ 开源

  • ✅ 点对点同步

  • ✅ 跨平台

备份策略

自动备份

使用文件恢复插件

设置 → 核心插件 → 文件恢复

  • 自动保存文件历史

  • 可以恢复到之前的版本

  • 保留时间可配置

使用第三方备份工具

  • Windows: Windows备份

  • macOS: Time Machine

  • Linux: rsync、Timeshift

手动备份

定期备份整个仓库：

# 复制整个仓库
cp -r /path/to/vault /path/to/backup/vault-$(date +%Y%m%d)

# 或使用rsync
rsync -av /path/to/vault/ /path/to/backup/

性能优化

大型仓库优化

如果你的仓库包含大量文件：

  1. 排除大文件

    设置 → 文件与链接 → 排除的文件

    添加：*.pdf, *.mp4, *.zip

  2. 禁用不需要的插件

    • 只启用必要的插件
    • 定期检查插件性能

  3. 使用文件夹

    • 合理组织文件结构
    • 避免单个文件夹文件过多

启动优化

  • 关闭不需要的核心插件

  • 减少社区插件数量

  • 使用工作区保存常用布局

常见问题

Q: Obsidian是否需要联网？

A: 不需要。Obsidian完全可以离线使用，所有数据存储在本地。

Q: 如何卸载Obsidian？

A:

  • Windows: 控制面板 → 程序和功能

  • macOS: 从Applications文件夹删除

  • Linux: 删除AppImage文件或使用包管理器卸载

Q: 仓库可以移动吗？

A: 可以。直接移动整个文件夹，然后在Obsidian中重新打开即可。

Q: 如何更新Obsidian？

A: Obsidian会自动检查更新。也可以在设置 → 关于中手动检查更新。
